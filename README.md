# Система бронирования репетиционной студии

Полнофункциональное веб-приложение для управления бронированием репетиционных залов с интеграцией Supabase.

## 🎯 Возможности

### Для клиентов:
- ✅ Регистрация и авторизация
- ✅ Просмотр доступных залов с фото и характеристиками
- ✅ Онлайн-бронирование с выбором даты, времени и длительности
- ✅ Управление своим профилем
- ✅ История бронирований

### Для администраторов:
- ✅ Управление всеми бронированиями
- ✅ Изменение статусов (ожидание, подтверждено, отменено)
- ✅ Удаление бронирований
- ✅ Статистика и отчёты
- ✅ Аналитика по залам и клиентам

## 🏗️ Технологический стек

### Frontend:
- **React 18** с TypeScript
- **React Router 7** для навигации
- **Material-UI 7** для UI компонентов
- **Recharts** для графиков и диаграмм
- **Tailwind CSS v4** для стилизации

### Backend:
- **Supabase** (Edge Functions + KV Store)
- **Hono** веб-фреймворк
- **Deno** runtime для Edge Functions

## 🚀 Быстрый старт

### 1. Запуск приложения

Приложение уже запущено и подключено к Supabase. Откройте браузер и перейдите к предпросмотру.

### 2. Регистрация пользователя

1. Нажмите "Зарегистрироваться"
2. Заполните форму:
   - Имя: Иван Петров
   - Email: test@example.com
   - Телефон: +7 999 123 45 67
   - Пароль: 123456
3. Нажмите "Зарегистрироваться"

### 3. Создание бронирования

1. Выберите один из доступных залов (A, B или C)
2. Укажите дату и время
3. Выберите длительность (1-4 часа)
4. Нажмите "Забронировать"
5. Проверьте детали и подтвердите

### 4. Просмотр статистики

Перейдите в раздел "Отчёты" для просмотра:
- Общего количества бронирований
- Общего дохода
- Среднего чека
- Распределения по залам
- Топ клиентов

## 🗄️ База данных

### Предустановленные залы:

| Зал   | Цена/час | Вместимость | Площадь |
|-------|----------|-------------|---------|
| Зал A | 1500 ₽   | до 5 чел    | 25 м²   |
| Зал B | 2000 ₽   | до 8 чел    | 40 м²   |
| Зал C | 1000 ₽   | до 3 чел    | 15 м²   |

### API Endpoints:

```
POST   /auth/register           # Регистрация
POST   /auth/login              # Вход
GET    /users/:id               # Профиль пользователя
PUT    /users/:id               # Обновить профиль
GET    /halls                   # Список залов
GET    /halls/:id               # Информация о зале
POST   /bookings                # Создать бронирование
GET    /bookings/:id            # Получить бронирование
GET    /bookings/user/:userId   # Бронирования пользователя
GET    /bookings                # Все бронирования
PUT    /bookings/:id            # Обновить бронирование
DELETE /bookings/:id            # Удалить бронирование
GET    /reports/statistics      # Статистика
```

## 🛠️ Разработка

### Добавление нового зала:

```typescript
// В supabase/functions/server/index.tsx
const newHall = {
  id: "hall-4",
  name: "Зал D",
  imageUrl: "url",
  capacity: 10,
  pricePerHour: 2500,
  area: 50,
  description: "Описание",
  isAvailable: true
};

await kv.set(`hall:${newHall.id}`, newHall);
```
